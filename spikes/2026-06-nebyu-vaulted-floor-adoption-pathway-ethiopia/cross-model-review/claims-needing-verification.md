## Purpose

This file lists claims, assumptions, analogues, and pathway inferences that should not be treated as accepted findings yet.

A claim appearing here does **not** mean the claim is false. It means the claim needs source cleanup, local review, or pathway testing before it is used in a final adoption-pathway brief or partner-facing material.

This file is especially important because both model generations use plausible institutional reasoning and analogies. Those are useful for research, but they must not become “facts” without verification.

## Verification Status Labels

Use these labels as the file is updated:

- **Open**: not yet checked
- **Partially supported**: some evidence exists, but not enough for final use
- **Supported**: evidence is adequate for internal synthesis
- **Partner-facing with caution**: usable externally only with careful wording
- **Needs local review**: requires Ethiopian engineer, contractor, university, permitting, or partner input
- **Do not use yet**: too uncertain or too high-risk to include
- **Retire**: unsupported, contradicted, or not useful

## Priority Labels

- **P0**: Must verify before final adoption-pathway brief
- **P1**: Should verify before partner-specific use
- **P2**: Useful but not blocking
- **P3**: Downstream / later-gate claim
