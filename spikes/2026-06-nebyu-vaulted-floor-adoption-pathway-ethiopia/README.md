# Nebyu Vaulted Floor Adoption Pathway in Ethiopia

This spike investigates the most plausible pathway by which Nebyu Samuel Haile’s vaulted earthen floor system could progress from research and testing toward a real, institutionally supported building in Ethiopia.

The focus is not whether the system should be adopted.

The focus is how construction-system innovations move forward when they do move forward:

- trust-building mechanisms
- host institutions
- evidence-generating building types
- adoption sequences
- coalition formation
- credibility transfer
- learning loops
- institutional incentives

This spike builds on earlier Pathfinder Research work, including:

- ribbed / HCB slab adoption in Ethiopia
- recent construction innovation pathways in Ethiopia
- Nebyu vaulted floor partner-readiness and first-pilot decision reality

## Core Question

Given the current state of evidence, what is the most plausible pathway by which Nebyu Samuel Haile’s vaulted earthen floor system could progress from research and testing into a real, institutionally supported building in Ethiopia?

## Scope

This is an innovation-adoption and institutional-pathway analysis.

It is not:

- a technology promotion memo
- a risk audit
- a commercialization plan
- an argument that the system should be adopted
